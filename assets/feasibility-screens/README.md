# Feasibility Study Screens

الصور التالية تم التقاطها من الواجهات الحالية للمشروع ويمكن استخدامها داخل دراسة الجدوى أو العرض التعريفي:

## الملفات

- `auth-login.png`
  - عنوان مقترح: شاشة الدخول والهوية البصرية للمنصة
  - استخدام: عرض الانطباع الأول وتجربة التسجيل والدخول.

- `user-dashboard.png`
  - عنوان مقترح: لوحة العميل لمتابعة القضايا والخدمات
  - استخدام: توضيح قيمة المنصة للمستخدم النهائي ومتابعة الحالة القانونية.

- `pro-dashboard.png`
  - عنوان مقترح: لوحة المحامي لإدارة القضايا والعملاء
  - استخدام: إبراز القيمة التشغيلية للمحامي أو المكتب القانوني.

- `ai-chat.png`
  - عنوان مقترح: المساعد القانوني الذكي باللغة العربية
  - استخدام: توضيح عنصر الذكاء الاصطناعي والمراجع القانونية داخل المنتج.

- `legal-library.png`
  - عنوان مقترح: مكتبة القوانين والبحث القانوني
  - استخدام: إبراز قاعدة المعرفة القانونية والبحث والتصفح.

## ملاحظة

هذه اللقطات مناسبة لدراسة الجدوى والعرض الأولي. وإذا رغبت يمكن لاحقًا تجهيز:

- نسخة بأسماء ملفات عربية.
- نسخة مضغوطة للعرض التقديمي.
- غلاف موحد يجمع 3 إلى 4 لقطات في صورة واحدة.
